# 整理券システム
整理券を生成・発券・抽選・管理することができるシステム

# 企画・イメージ・要件

* [調整さん](https://chouseisan.com/) みたいな感じに気軽に作れて気軽に共有できて、気軽に抽選ができるような感じ
* 気軽にイベントを立てられる。管理用と応募用にURLがイベントごとに立てられる。
* 整理券を実際の紙で印刷できるように番号とQRコー
*
